本代码基于深度学习 LSTM 模型，对单一股票股价进行预测。
1.  './data/' 可存放股票历史价格信息。其信息可在yahoo财经上进行直接下载
2. './result' 为训练结果。仅供参考。
3. 可修改步长，层数等参数。
